# Tiki Desktop

A desktop application for the [Tiki](https://github.com/Eric-Ness/Tiki) workflow framework. Provides a visual interface for managing Claude Code workflows with terminal integration, workflow visualization, and GitHub integration.

## Features (Planned)

- **Multi-terminal management** - Run multiple Claude Code sessions
- **Workflow visualization** - n8n-style diagram showing phase execution
- **GitHub integration** - Issue management, releases, milestones
- **Real-time state** - Live updates from Tiki execution
- **Project management** - Switch between multiple projects

## Tech Stack

- Electron
- React + TypeScript
- React Flow (workflow diagrams)
- xterm.js (terminal emulation)
- Tailwind CSS + shadcn/ui

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```

## Status

This project is in early development. Using Tiki to build itself!
